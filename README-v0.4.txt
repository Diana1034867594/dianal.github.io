HOW TO INSTALL
1. Open GitHub Desktop.
2. Choose Repository > Show in Explorer.
3. Copy the three items from this update package into the repository root:
   projects.html
   css folder
   js folder
4. Choose Replace files in the destination.
5. Return to GitHub Desktop.
6. Commit summary: Redesign projects directory
7. Click Commit to main, then Push origin.
8. After deployment, hard refresh with Ctrl + F5.
