Tutelarius Simulator update — Version 0.5

This package adds:
- a complete Tutelarius Simulator case-study page

- the Panopto demonstration embedded responsively

INSTALL
1. Copy every item inside this folder into the root of your cloned repository.
2. Choose Replace when prompted.
3. In GitHub Desktop, commit with: Add Tutelarius case study assets
4. Push origin.

Important: copy the whole assets folder, not only the HTML page.
